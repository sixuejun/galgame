# Eden System - 组件使用指南

## 1. 概述

Eden System 提供了一套可复用的通用组件，位于 `src/components/common/` 目录。这些组件经过精心设计，具有良好的可访问性、类型安全和测试覆盖。

## 2. 图标组件

### Icon

**位置**: `src/components/common/Icon.vue`

通用图标组件，支持多种图标类型。

**Props**:

| 属性         | 类型   | 默认值         | 说明                                           |
| ------------ | ------ | -------------- | ---------------------------------------------- |
| iconData     | string | -              | 图标数据（Font Awesome 类名、图片 URL 或文本） |
| alt          | string | '图标'         | 图标的替代文本                                 |
| fallbackIcon | string | 'fas fa-image' | 图片加载失败时的备用图标                       |

**使用示例**:

```vue
<template>
  <!-- Font Awesome 图标 -->
  <Icon iconData="fas fa-star" alt="星标" />

  <!-- 图片 URL -->
  <Icon iconData="https://example.com/icon.png" alt="自定义图标" />

  <!-- 纯文本图标（Emoji） -->
  <Icon iconData="⭐" alt="星星" />
</template>

<script setup lang="ts">
import Icon from '@/components/common/Icon.vue'
</script>
```

**特性**:

- 自动识别图标类型（Font Awesome、URL、文本）
- 图片加载失败时显示备用图标
- 无障碍支持（ARIA 属性）

## 3. 进度条组件

### ProgressBar

**位置**: `src/components/common/ProgressBar.vue`

进度条组件，显示进度百分比。

**Props**:

| 属性     | 类型   | 默认值 | 说明                 |
| -------- | ------ | ------ | -------------------- |
| label    | string | -      | 标签文本             |
| current  | number | -      | 当前值               |
| max      | number | -      | 最大值               |
| barClass | string | -      | 进度条样式类名或颜色 |

**使用示例**:

```vue
<template>
  <!-- 使用预定义样式 -->
  <ProgressBar label="经验值" :current="500" :max="1000" barClass="exp-bar" />

  <!-- 使用自定义颜色 -->
  <ProgressBar label="生命值" :current="80" :max="100" barClass="#ff4444" />

  <!-- 使用渐变 -->
  <ProgressBar
    label="魔法值"
    :current="60"
    :max="100"
    barClass="linear-gradient(90deg, #667eea 0%, #764ba2 100%)"
  />
</template>

<script setup lang="ts">
import ProgressBar from '@/components/common/ProgressBar.vue'
</script>
```

**特性**:

- 动画效果（平滑过渡）
- 支持 CSS 变量、颜色值、渐变
- 显示当前值/最大值
- 条纹动画效果

## 4. 输入组件

### CustomNumberInput

**位置**: `src/components/common/CustomNumberInput.vue`

自定义数字输入组件，带增减按钮。

**Props**:

| 属性       | 类型   | 默认值 | 说明               |
| ---------- | ------ | ------ | ------------------ |
| modelValue | number | -      | 当前值（v-model）  |
| min        | number | 0      | 最小值             |
| max        | number | 100    | 最大值             |
| step       | number | 1      | 步长               |
| label      | string | ''     | 标签（用于无障碍） |

**Emits**:

- `update:modelValue(value: number)`: 值变化时触发

**使用示例**:

```vue
<template>
  <CustomNumberInput v-model="quantity" :min="1" :max="99" :step="1" label="购买数量" />
</template>

<script setup lang="ts">
import { ref } from 'vue'
import CustomNumberInput from '@/components/common/CustomNumberInput.vue'

const quantity = ref(1)
</script>
```

**特性**:

- 增加/减少按钮
- 最小值/最大值限制
- 输入验证
- 无障碍支持

### CustomSelect

**位置**: `src/components/common/CustomSelect.vue`

自定义下拉选择组件。

**Props**:

| 属性        | 类型                                  | 默认值   | 说明                  |
| ----------- | ------------------------------------- | -------- | --------------------- |
| modelValue  | string                                | -        | 当前选中值（v-model） |
| options     | Array<{value: string, label: string}> | -        | 选项列表              |
| placeholder | string                                | '请选择' | 占位符                |
| label       | string                                | ''       | 标签（用于无障碍）    |

**使用示例**:

```vue
<template>
  <CustomSelect
    v-model="selectedCategory"
    :options="categoryOptions"
    placeholder="选择分类"
    label="物品分类"
  />
</template>

<script setup lang="ts">
import { ref } from 'vue'
import CustomSelect from '@/components/common/CustomSelect.vue'

const selectedCategory = ref('')
const categoryOptions = [
  { value: 'weapon', label: '武器' },
  { value: 'armor', label: '防具' },
  { value: 'consumable', label: '消耗品' },
]
</script>
```

## 7. 分页组件

### Pagination

**位置**: `src/components/common/Pagination.vue`

分页组件。

**Props**:

| 属性            | 类型   | 默认值 | 说明             |
| --------------- | ------ | ------ | ---------------- |
| currentPage     | number | 1      | 当前页码         |
| totalPages      | number | -      | 总页数           |
| maxVisiblePages | number | 5      | 最多显示的页码数 |

**Emits**:

- `page-change(page: number)`: 页码变化时触发

**使用示例**:

```vue
<template>
  <Pagination
    :current-page="currentPage"
    :total-pages="totalPages"
    :max-visible-pages="7"
    @page-change="handlePageChange"
  />
</template>

<script setup lang="ts">
import { ref } from 'vue'
import Pagination from '@/components/common/Pagination.vue'

const currentPage = ref(1)
const totalPages = ref(10)

function handlePageChange(page: number) {
  currentPage.value = page
  // 加载新页面数据
}
</script>
```

## 9. 最佳实践

### 9.1 组件导入

使用路径别名导入组件：

```typescript
// ✅ 好：使用 @ 别名
import Icon from '@/components/common/Icon.vue'

// ❌ 不好：使用相对路径
import Icon from '../../../components/common/Icon.vue'
```

### 9.2 Props 验证

充分利用 TypeScript 类型检查：

```vue
<script setup lang="ts">
import { ref } from 'vue'
import CustomNumberInput from '@/components/common/CustomNumberInput.vue'

// TypeScript 会检查 props 类型
const quantity = ref(1)
</script>

<template>
  <!-- 类型错误会在编译时被捕获 -->
  <CustomNumberInput v-model="quantity" :min="1" :max="99" />
</template>
```

### 9.3 无障碍支持

为组件提供适当的 ARIA 属性：

```vue
<template>
  <Icon iconData="fas fa-star" alt="收藏" />
  <CustomNumberInput v-model="quantity" label="购买数量" />
</template>
```

### 9.4 样式定制

使用 CSS 变量定制组件样式：

```vue
<style>
:root {
  --primary-blue: #0080ff;
  --progress-exp: #4caf50;
  --progress-hp: #f44336;
}
</style>
```

## 10. 测试


**测试示例**:

```typescript
// CustomNumberInput.test.ts
import { describe, it, expect } from 'vitest'
import { mount } from '@vue/test-utils'
import CustomNumberInput from '../CustomNumberInput.vue'

describe('CustomNumberInput', () => {
  it('应该正确渲染', () => {
    const wrapper = mount(CustomNumberInput, {
      props: { modelValue: 5, min: 1, max: 10 },

    expect(wrapper.find('.number-input').element.value).toBe('5')
  })
})
```
