# 功能说明文档

> 末日报纸风 Galgame 流式楼层界面 — 功能说明与模块清单

---

## 一、整体架构

### 技术栈

- **Vue 3** + **Pinia** 状态管理
- **Tailwind CSS v4** + 自定义 CSS 变量（`theme.css`）
- **流式楼层界面**：基于 `mountStreamingMessages` 挂载，每个 AI 消息楼层独立 iframe Vue 实例
- 持久化：设置存于 `script` 变量，游戏数据存于 `chat` 变量

### 视觉风格

- 复古做旧报纸质感：泛黄纸张、褪色油墨、折痕污渍
- 色调：灰暗土黄 `#d4c5a0`、灰黑 `#2a2420`、锈色 `#8b4513`
- 16:9 固定比例视口，报纸装饰元素（EXTRA EDITION / 末日旧闻 / 第壹百零七期）

### 图层结构（从下到上）

1. **背景层**：渐变 + 报纸标题水印
2. **弹幕层**：screen-space，不受场景抖动影响
3. **暗角层**：径向渐变暗角
4. **噪点层**：SVG fractalNoise 纹理
5. **UI 层**（z-index 20）：快捷菜单 + 对话框
6. **选项层**：玩家选择面板
7. **覆盖面板**（z-index 50）：设置/历史/角色/玩法
8. **模块面板**（z-index 55）：各玩法模块 overlay
9. **系统通知**（z-index 9999）：Toast 弹窗，永远在最上层

## 二、核心界面组件

### 2.1 对话框 `DialogueBox`

- 打字机效果逐字显示
- 前后翻页按钮
- 说话人名称 / 旁白指示
- 用户头像可选显示

### 2.2 选择面板 `ChoicePanel`

- 预定义选项 + 自定义输入
- 选择后短暂延迟提交（`/send` 命令）
- 纸飞机图标发送自定义文本

### 2.3 快捷菜单 `QuickAccessMenu`

- 左上：角色、玩法
- 右上：设置、全屏、历史
- 胶囊按钮展开/折叠，点击外部自动收起
- **全屏**：请求 iframe 元素全屏显示

### 2.4 舞台区 `StageArea`

- 大气背景渐变 + 报纸标题装饰
- 弹幕层（可配置 1/3 屏 / 半屏 / 全屏）
- 暗角 + 噪点叠加

## 三、覆盖面板

### 3.1 设置面板 `SettingsPanel`

| 区域     | 内容                                                               |
| -------- | ------------------------------------------------------------------ |
| 音量设置 | BGM / 音效 / 语音滑块                                              |
| 文字显示 | 速度 / 自动速度 / 自动播放开关                                     |
| 弹幕设置 | 启用开关 / 速度 / 循环 / 显示区域（1/3屏/半屏/全屏）               |
| 显示设置 | 竖屏模式开关                                                       |
| API 配置 | 第二 API（URL + Key + 测试连接）/ 生图 API（URL + Key + 测试连接） |
| 界面皮肤 | 旧报 / 电报 / 战时 / 档案                                          |
| 恢复默认 | 一键重置所有设置                                                   |

- API Provider 状态：`available`（已连接）/ `disabled`（未配置）
- 测试按钮发送 GET 请求并显示连接结果

### 3.2 历史面板 `HistoryPanel`

- 报纸栏目式排版
- 行号 + 说话人 + 旁白缩进
- 高亮当前活跃行

### 3.3 角色面板 `CharacterPanel`

- 用户角色：头像上传 / 名称编辑 / 头像显示开关
- 角色花名册：展示解锁状态、好感度、描述

### 3.4 玩法面板 `GameplayPanel`

**首栏信息：**

- 金币显示（实时）
- 背包快捷入口（物品数量标记）
- 工坊等级显示

**模块卡片网格：**

- 动态锁定：商店和情报交换在第二 API 未配置时自动灰掉
- 工坊卡片上显示生产状态（生产中 / 已暂停），带快速暂停/启动图标按钮
- 2048 卡片标注本次费用
- 点击模块卡片打开对应模块

## 四、经济系统 `EconomyService`

- **统一走账**：所有金币变更必须通过 `changeGold(amount, moduleId, reason)`
- **交易日志**：最多保留 50 条记录（模块ID / 原因 / 数额 / 时间戳）
- **背包系统**：`addInventoryItem` 支持堆叠
- **抽成公式**：`calcCommission(gold, workshopLevel, basePrice, threshold=500)`
  - 金币 ≤ 阈值时仅收取基价
  - 超过阈值按工坊等级 0.5%/级收取，封顶 5%
- **股票手续费**：`calcStockFee(workshopLevel)` = workshopLevel × 5，封顶 50

## 五、玩法模块

### 5.1 背包 `InventoryModule`

- 物品列表（名称 / 效果 / 数量）
- 总物品数统计

### 5.2 金币记录 `GoldLogModule`

- 交易日志列表（模块标签 / 原因 / 时间 / 金额增减）
- 当前余额显示

### 5.3 商店 `ShopModule`

- **解锁条件**：需配置第二 API
- 付费刷新（基价 50G + 抽成），携带 generationId 防并发
- 购买即时扣款、物品即时入背包
- 刷新中禁用按钮

### 5.4 工坊 & 交易 `WorkshopModule`（合并工坊 + 股市）

#### 工坊生产

- 工坊等级 1-10，每级 +10% 产出加成
- 选择已解锁角色开始生产
- 按真实时间每秒计算即时增加（离线不算）
- **快速暂停/继续**：暂停时累计已产出，继续时从暂停点继续
- 停止结算：一次性计算差值入账
- 升级消耗：等级 × 200G

#### 风险与机遇（股市）

- **与生产并行**：经营股票不打断生产
- **现金同步**：可支配现金即玩家金币（允许生产收入实时增加可用金额）
- **价格模拟**：离散随机游走 + 轻微均值回归 + 交易冲击
  - 3 秒一次自动变化，交易后立即变化
  - 暂停时折线不变化、无法交易
- **操作**：
  - 买入（+）：按当前价 + 手续费 扣除金币，持仓 +1
  - 卖出（-）：持仓 -1，按当前价 - 手续费 返还金币
  - 可连点快速操作
- **折线图**：
  - 浅色背景 + 水平/垂直网格线 + 价格标签
  - 涨跌用绿色/红色箭头标注
  - 渐变填充（涨绿 / 跌红）
  - 退出（返回）时自动重置
  - 使用 ResizeObserver + nextTick 确保渲染
- **刷新**：随机生成不同初始价格开局
- **暂停/继续**：图标按钮控制
- **退出**：自动平仓卖出全部持仓
- **预计收益**：实时计算（持仓价值 - 手续费 - 已投入）

### 5.5 思绪整理 `Puzzle2048Module`（2048 合成小游戏）

- **点开即玩**：点击卡片自动扣费并开始
- **费用**：金币 ≤ 500 免费，超过按抽成公式计算
- **动态棋盘**：
  - 默认 4×4
  - 4 级工坊解锁 6×6
  - 8 级工坊解锁 8×8
  - 有多个可选大小时显示选择器
- 标准 2048 规则，方块封顶 2048
- 键盘（WASD / 方向键）+ 触屏滑动操作
- **收益**：方块数字之和 × 2048 倍率
  - 0 个 2048: ×1.0
  - 1 个 2048: ×1.5
  - 2 个 2048: ×2.2
  - 3+ 个 2048: ×3.0
- 保存退出：保留棋盘、不结算
- 结算退出：清空棋盘、收益入账

### 5.6 情报交换 `RiddleModule`（AI 对话猜谜）

- **解锁条件**：需配置第二 API
- 用户输入谜底（本地校验，不进入 prompt）
- 用户输入第一条提示开始游戏
- 对话式猜谜，AI 逐轮回复
- **谜底屏蔽**：用户输入包含谜底时阻止发送
- **胜利条件**：AI 回复匹配到谜底
- **奖励**：50 + 轮数 × 20 金币
- 保留上次战绩记录

## 六、弹幕层 `DanmakuLayer`

- 位于 Stage 顶部，默认占 1/3 区域（可配置）
- screen-space 坐标，不受相机抖动/缩放影响
- 不遮挡对话框 / 选项区 / 快捷入口
- 密度控制：队列上限 20 条，超出丢弃
- 可在设置中开关、调整速度、切换循环和显示区域
- 数据来源：第二 API

---

## 七、API Provider 系统

| Provider | 用途                         | 失败策略                 |
| -------- | ---------------------------- | ------------------------ |
| 第二 API | 商店刷新、弹幕生成、猜谜对话 | 相关模块自动灰掉不可解锁 |
| 生图 API | 背景或 CG 生成               | 功能禁用                 |

- Provider 状态：`available` / `disabled`
- 设置界面提供 URL + Key 输入 + 测试连接按钮
- 模块读取 Provider 状态自动灰掉/提示

---

## 八、跨模块共通规则

- 所有玩法模块不影响主剧情状态、对话指针、分支锁定
- 金币/资源统一进入 EconomyService
- 同一时间最多打开一个玩法模块
- 退出工坊前若正在生产，阻止关闭（需先停止结算或暂停）
- 系统通知 Toast 永远在最上层（z-index 9999）
