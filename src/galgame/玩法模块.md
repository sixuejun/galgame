## 一、经济系统

### EconomyService

- **统一走账**：所有金币变更必须通过 `changeGold(amount, moduleId, reason)`
- **交易日志**：最多保留 50 条记录（模块ID / 原因 / 数额 / 时间戳）
- **背包系统**：`addInventoryItem` 支持堆叠
- **抽成公式**：`calcCommission(gold, workshopLevel, basePrice, threshold=500)`
  - 金币 ≤ 阈值时仅收取基价
  - 超过阈值按工坊等级 0.5%/级收取，封顶 5%
- **股票手续费**：`calcStockFee(workshopLevel)` = workshopLevel × 5，封顶 50

## 二、玩法模块

### 2.1 背包 `InventoryModule`

- 物品列表（名称 / 效果 / 数量）
- 总物品数统计

### 2.2 金币记录 `GoldLogModule`

- 交易日志列表（模块标签 / 原因 / 时间 / 金额增减）
- 当前余额显示

### 2.3 商店 `ShopModule`

- **解锁条件**：需配置第二 API
- 付费刷新（基价 50G + 抽成），携带 generationId 防并发
- 购买即时扣款、物品即时入背包
- 刷新中禁用按钮

### 2.4 工坊 & 交易 `WorkshopModule`（合并工坊 + 股市）

#### 工坊生产

- 工坊等级 1-10，每级 +10% 产出加成
- 选择已解锁角色开始生产
- 按真实时间每秒计算即时增加（离线不算）
- **快速暂停/继续**：暂停时累计已产出，继续时从暂停点继续
- 停止结算：一次性计算差值入账
- 升级消耗：等级 × 200G

#### 风险与机遇（股市）

- **与生产并行**：经营股票不打断生产
- **现金同步**：可支配现金即玩家金币（允许生产收入实时增加可用金额）
- **价格模拟**：离散随机游走 + 轻微均值回归 + 交易冲击
  - 3 秒一次自动变化，交易后立即变化
  - 暂停时折线不变化、无法交易
- **操作**：
  - 买入（+）：按当前价 + 手续费 扣除金币，持仓 +1
  - 卖出（-）：持仓 -1，按当前价 - 手续费 返还金币
  - 可连点快速操作
- **折线图**：
  - 浅色背景 + 水平/垂直网格线 + 价格标签
  - 涨跌用绿色/红色箭头标注
  - 渐变填充（涨绿 / 跌红）
  - 退出（返回）时自动重置
  - 使用 ResizeObserver + nextTick 确保渲染
- **刷新**：随机生成不同初始价格开局
- **暂停/继续**：图标按钮控制
- **退出**：自动平仓卖出全部持仓
- **预计收益**：实时计算（持仓价值 - 手续费 - 已投入）

### 2.5 思绪整理 `Puzzle2048Module`（2048 合成小游戏）

- **点开即玩**：点击卡片自动扣费并开始
- **费用**：金币 ≤ 500 免费，超过按抽成公式计算
- **动态棋盘**：
  - 默认 4×4
  - 4 级工坊解锁 6×6
  - 8 级工坊解锁 8×8
  - 有多个可选大小时显示选择器
- 标准 2048 规则，方块封顶 2048
- 键盘（WASD / 方向键）+ 触屏滑动操作
- **收益**：方块数字之和 × 2048 倍率
  - 0 个 2048: ×1.0
  - 1 个 2048: ×1.5
  - 2 个 2048: ×2.2
  - 3+ 个 2048: ×3.0
- 保存退出：保留棋盘、不结算
- 结算退出：清空棋盘、收益入账

### 2.6 情报交换 `RiddleModule`（AI 对话猜谜）

- **解锁条件**：需配置第二 API
- 用户输入谜底（本地校验，不进入 prompt）
- 用户输入第一条提示开始游戏
- 对话式猜谜，AI 逐轮回复
- **谜底屏蔽**：用户输入包含谜底时阻止发送
- **胜利条件**：AI 回复匹配到谜底
- **奖励**：50 + 轮数 × 20 金币
- 保留上次战绩记录
- **提示词方案**：不使用预设中的 EJS，直接在代码中构造包含角色设定和猜谜规则的提示词
  - 包含示例对话帮助 AI 理解交互模式
  - 支持自定义每轮回复长度建议

## 三、废土行路

### 3.1 格子分布与类型

地图上的格子（节点）共有 **5 种类型**，格子类型仅作为"结果倾向权重"：

| 类型 | 英文名 | 占比 | 含义 | 结果倾向权重 |
|------|--------|------|------|-------------|
| **起点** | `start` | 1个 | 地图起始位置 | 无事件 |
| **空地** | `empty` | 40% | 安全区域 | 无事件 |
| **遭遇** | `encounter` | 35% | 遇到陌生人、NPC 等 | 所有倾向等概率 |
| **陷阱** | `trap` | 15% | 地雷、毒气、坍塌等危险 | 偏负面 (60% 负面) |
| **意外之喜** | `fortune` | 10% | 补给站、发现宝物等好事 | 偏正面 (60% 正面) |

**终点**：`end`，到达后地图刷新重来。

### 3.2 统一事件池

所有事件统一存储在 `eventsConfig.ts` 中，每张卡牌都带有 `tendency` 属性：

| 倾向 | 英文名 | 含义 |
|------|--------|------|
| **负面** | `negative` | 偏损失，如危险区域、误触装置、埋伏、污染等 |
| **正面** | `positive` | 偏收益，如补给、捷径、情报、成功逃脱等 |
| **中性** | `neutral` | 带随机性或风险交换，如遇见新的人物、异常现象、对峙等 |

### 3.3 卡牌效果

卡牌 `effect` 包含以下属性：

| 属性 | 说明 |
|------|------|
| `hp` | 生命值变化（正数恢复，负数扣除） |
| `sanity` | 精神值变化（正数恢复，负数扣除） |
| `transfer` | 是否传送（传送到随机空地） |
| `message` | 效果描述文本 |

**注意**：卡牌效果不再直接调整"运气"，但 MVU 变量中的运气值会影响事件抽取概率。

### 3.4 运气影响

玩家可以通过 MVU 变量设置个人属性中的"运气"值：

```yaml
个人属性:
  运气: 5  # 基准值为5
  体质: 5
  耐力: 5
  特殊能力: ['异度空间']
```

运气值会影响事件抽取：
- 运气 > 5：高运气增加正面事件概率，减少负面事件概率
- 运气 < 5：低运气则相反
- 影响幅度为小幅度（每点运气约 ±5%）

### 3.5 事件抽取流程

```
步数用完 → 根据格子类型确定倾向权重 → 应用运气修正 → 抽取事件 → 显示卡牌选择
```

**各格子类型的倾向权重**：

| 格子类型 | 负面 | 中性 | 正面 |
|----------|------|------|------|
| `encounter` | 33% | 33% | 33% |
| `trap` | 60% | 25% | 15% |
| `fortune` | 15% | 25% | 60% |

### 3.6 AI 事件生成

当启用 AI 事件生成时，系统会：
1. 优先使用 AI 预生成的事件
2. AI 返回 9-12 张事件卡，每张包含 `title`、`description`、`tendency`、`effect`、`hp`、`sanity`
3. 系统将这些卡片分组为 2-3 个完整事件（每个事件 2-4 张卡供选择）
4. 如果 AI 生成失败，自动降级使用本地事件池

### 3.7 已实现的功能

#### 鼠标移动到地图边缘时自动移动视图

**功能描述：**
- 当鼠标靠近地图视图边缘（80像素范围内）时，会显示方向箭头提示
- 鼠标悬停在箭头上时，视图会自动向该方向平移
- 支持上下左右四个方向的自动滚动

**实现细节：**
- 添加了 4 个方向箭头组件（上、下、左、右）
- 使用 `handleMouseMove` 函数检测鼠标位置
- 边缘阈值：80 像素
- 滚动速度：每帧 8 像素
- 箭头带有呼吸动画效果，提升视觉体验

#### 投掷骰子后自动移动

**功能描述：**
- 投掷骰子后，如果只有一条路径可走，系统会自动移动
- 遇到分叉路口（多条路径）时停下，等待玩家选择
- 自动移动过程中会显示剩余步数

**逻辑流程：**
1. 玩家掷骰子
2. 系统检查可走路径数量
3. 如果只有 1 条路径 → 自动移动
4. 如果有 2 条或以上路径 → 停下等待玩家选择
5. 重复步骤 2-4 直到步数用完

#### 步数走完之前不进入随机事件

**功能描述：**
- 只有在步数完全用完（`stepsRemaining <= 0`）时才会触发事件
- 在自动移动过程中，即使经过事件格子也不会触发
- 确保玩家能完整走完骰子点数

#### 视图自动跟随玩家

**额外优化功能：**
- 玩家移动时，视图会平滑地跟随玩家位置
- 使用 GSAP 动画库实现流畅的视图过渡
- 在动画过程中实时更新视口位置
- 移动完成后自动居中到玩家位置

```typescript:store.ts
// 创建股票模拟器
export function createStockSimulator(initialPrice = 100) {
  let price = initialPrice;
  let drift = 0;    // 漂移项
  let impact = 0;   // 交易冲击
  let anchorPrice = initialPrice; // 锚定价格（用于均值回归）

  function tick() {
    // 1. 漂移项更新
    drift += (Math.random() - 0.5) * 0.3;
    drift = Math.max(-2, Math.min(2, drift)) * 0.95;

    // 2. 随机噪声
    const noise = (Math.random() - 0.5) * 4;

    // 3. 均值回归
    const reversion = (anchorPrice - price) * 0.02;

    // 4. 交易冲击衰减
    impact *= 0.85;

    // 5. 更新价格
    price = Math.max(1, price + drift + noise + reversion + impact);

    return Math.round(price * 100) / 100;
  }

  return { tick, applyTradeImpact, getPrice, getHistory, reset };
}

// 交易影响
function applyTradeImpact(direction: 'buy' | 'sell', amount: number) {
  impact += direction === 'buy' ? amount * 0.1 : -amount * 0.1;
}

// 买入流程
function stockBuy() {
  const cost = Math.ceil(stockPrice.value);  // 股价
  const fee = calcStockFee(workshopLevel);     // 手续费
  if (gold < cost + fee) return false;
  changeGold(-(cost + fee), 'stock_market', '买入 1 股');
  stockPosition.value += 1;
  stockInvested.value += cost;
  stockSimulator.applyTradeImpact('buy', 1); // 买入推高价格
  doStockTick(); // 立即更新价格
}

// 卖出流程
function stockSell() {
  const revenue = Math.floor(stockPrice.value);
  const fee = calcStockFee(workshopLevel);
  changeGold(revenue - fee, 'stock_market', '卖出 1 股');
  stockPosition.value -= 1;
  stockSimulator.applyTradeImpact('sell', 1); // 卖出压低价格
  doStockTick();
}

// 退出（自动平仓）
function exitStockMarket() {
  stopStockTicker();
  while (stockPosition.value > 0) {
    const revenue = Math.floor(stockPrice.value);
    const net = revenue - calcStockFee(workshopLevel);
    changeGold(net, 'stock_market', '平仓卖出');
    stockPosition.value--;
  }
  resetStock();
}
```

```typescript:store.ts
async function refreshShop() {
  // 1. 检查第二 API 状态
  if (secondApiStatus.value === 'disabled') {
    showToast('第二 API 未配置');
    return;

  }
  // 扣款
  changeGold(-item.price, 'shop', `购买 ${item.name}`);
  // 入背包
  addInventoryItem({ id: item.id, name: item.name, effect: item.effect });
  // 移除已购商品
  shopItems.value = shopItems.value.filter(i => i.id !== itemId);
  return true;
}
```

### 6. 废土行路状态机

```typescript:boardgame/boardGameStore.ts
// 游戏阶段
export type GamePhase = 'idle' | 'rolling' | 'choosingPath' | 'event' | 'resolving';

// 状态机转换
function rollDice() {
  // idle → rolling
  phase.value = 'rolling';
  diceValue.value = Math.floor(Math.random() * 6) + 1;
}

function finishRoll() {
  // rolling → choosingPath
  stepsRemaining.value = diceValue.value;
  phase.value = 'choosingPath';
  updateWalkable(); // 计算可走路径
}

function moveToNode(targetId: string) {
  // choosingPath → (choosingPath 或 event 或 idle)
  currentNodeId.value = targetId;
  stepsRemaining.value -= 1;
  moveHistory.value.push(targetId);
  isAnimating.value = true;
  return { from, to: targetId };
}

function onMoveAnimationDone() {
  // 动画完成后处理
  if (node.type === 'end') {
    // 到达终点 → 刷新地图
    regenerateMap(randomSeed);
    return;
  }

  if (stepsRemaining.value <= 0) {
    // 步数用完 → 触发事件
    if (node.type !== 'empty' && node.type !== 'start') {
      triggerEvent(node);
    } else {
      phase.value = 'idle';
    }
  } else {
    // 步数未用完 → 更新可走路径
    updateWalkable();
  }
}

function triggerEvent(node: MapNode) {
  // choosingPath | idle → event
  const event = getRandomEvent(node.type);
  if (!event) {
    // 空地或无事件 → 停在原地
    addLog(`□ 停在空地`);
    phase.value = 'idle';
    return;
  }
  currentEvent.value = event;
  phase.value = 'event';
  addLog(`⚡ 触发事件：${event.title}`);
}

function selectCard(card: EventCard) {
  // event → resolving
  selectedCard.value = card;
  phase.value = 'resolving';
  applyCardEffect(card); // 应用卡牌效果
}

function finishResolve() {
  // resolving → idle
  phase.value = 'idle';
  currentEvent.value = null;
  selectedCard.value = null;
}
```

```typescript:store.ts
// 商店刷新费用
export function calcCommission(gold, workshopLevel, basePrice, threshold = 500): number {
  if (gold <= threshold) return basePrice;  // 金币低于阈值，只收基价
  const rate = Math.min(0.05, workshopLevel * 0.005); // 0.5% × 等级，最多 5%
  return basePrice + Math.floor(gold * rate);
}

// 股票手续费
export function calcStockFee(workshopLevel): number {
  return Math.min(50, workshopLevel * 5); // 5 × 等级，最多 50
}

export function get2048Size(workshopLevel): number[] {
  const sizes = [4];           // 默认 4×4
  if (workshopLevel >= 4) sizes.push(6); // 4级解锁 6×6
  if (workshopLevel >= 8) sizes.push(8); // 8级解锁 8×8
  return sizes;
}
```
